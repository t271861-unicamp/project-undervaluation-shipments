# project-undervaluation-shipments
Repository containing the files used in the article entitled: Undervaluation of goods in cross-border e-commerce: A fraud detection approach

# Project Description

The project aims to develop a approach to support the detection of undervaluation fraud in international shipments, using Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG) techniques.

The proposed approach includes:
- Synthetic generation of Shipment Import Declarations (DIRs) using LLMs. 
- Construction of a synthetic vector database of products for comparative purposes.
- Application of the RAG technique in the declared value verification process.  
- Mapping of the operational inspection process through interviews with stakeholders from the public and private sectors.

└── README.md # This file
